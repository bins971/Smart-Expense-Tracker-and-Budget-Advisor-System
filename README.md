
## 🔧 Tech Stack

**Frontend:**
- React.js
- Material-UI (MUI)
- Axios
- React Router
- Chart.js

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Groq AI SDK

## 📝 Available Scripts

**Backend:**
- `npm start` - Start backend server with nodemon

**Frontend:**
- `npm start` - Start development server
- `npm run build` - Build for production
